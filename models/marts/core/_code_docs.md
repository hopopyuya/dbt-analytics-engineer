{% docs is_order_completed_docblock %}
注文が完了したかどうかを示す二進データです。注文のステータスを考慮して、以下のいずれかの値が含まれる可能性があります：

| is_order_completed | 定義 |
| ------------------ | ------------------------------------------------- |
| 1                  | ステータスに基づき、すでに完了している注文 |
| 0                  | ステータスに基づき、まだ完了していない注文 |
